### CPU
<a id=CPU></a>
#### 1. ​컴퓨팅의 물리적 임계점 (에너지와 양자)
**​모든 연산은 공짜가 아닙니다.**

CPU 설계의 시작은 전자가 움직이는 물리적 제약 조건을 극복하는 것에서 시작됩니다.

​랜다우어 한계 (Landauer’s Principle): 정보 1비트를 삭제하거나 변경할 때 발생하는 최소한의 열에너지입니다(E ≥ kT ln(2)).

즉, **CPU**가 빨라질수록 **열이 발생하는 것은 피할 수 없는 물리 법칙입니다**.

​준-발리스틱 전송 (Quasi-Ballistic Transport): 2nm 이하 공정에서는 전자가 장애물(원자)에 부딪히지 않고 마치 총알처럼 날아가는 현상이 발생합니다. 
이를 통제하는 것이 최신 공정의 핵심입니다.

​양자 터널링 (Quantum Tunneling): 트랜지스터가 너무 작아져서 전자가 막혀있는 벽을 뚫고 지나가 버리는 현상입니다. 
이를 막기 위해 GAA(Gate-All-Around) 구조가 도입되었습니다.
