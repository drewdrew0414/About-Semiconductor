## PART 1

### 1. 컴퓨팅의 제로 레벨: 열역학과 정보
* #### 1-1. 랜다우어 한계:
  정보 삭제 1비트당 최소 에너지
  ```
  E ≥ kT ln(2)

  ​E (에너지): 정보를 처리(삭제)할 때 소비되는 최소 에너지
  ​k (볼츠만 상수): 입자 하나하나의 미시적인 에너지를 온도와 연결해 주는 상수.
  ​T (절대 온도): 연산이 일어나는 환경의 온도(단위: 켈빈, K).
  ​ln(2): 자연로그 2를 의미하며, 정보의 기본 단위인 '1비트'가 두 가지 상태(0 또는 1)를 가질 수 있다는 점에서 유도됨. (약 0.693)
  ```
  * 이 수식이 말하고자 하는 핵심
​    * 현대 컴퓨팅 기술에서 이 법칙이 중요한 이유는 세 가지다.

    * ​정보는 물리적이다: 정보는 단순히 논리적인 개념이 아니라, 그것을 기록하고 지우는 과정에서 반드시 물리적인 에너지를 소모

​    * 발열의 근본적 이유: 아무리 효율적인 컴퓨터를 만들어도, 1비트의 정보를 지울 때 발생하는 에너지가 열로 바뀐다. 즉, **"공짜 연산은 없다"**는 뜻

    * ​컴퓨팅의 물리적 한계: 반도체 기술이 발전하면서 트랜지스터 전력 소모를 계속 줄여왔지만, 결국 이 '랜다우어 한계'에 도달하면 더 이상 에너지 효율을 높이는 것이 불가능하다.
